# Nutrients Almighty

## 프로젝트 소개
- Nutrients Almighty은 영양제 판매 사이트입니다.
- 해당 프로젝트는 KH정보교육원의 풀스택 웹개발자 과정을 진행중인 5명의 수강생이 설계 및 개발을 하였습니다.

## 내가 맡은 기능
- 사용자페이지 - 고객후기 페이지와 기능
- 관리자페이지 - FAQ, 공지사항, 후기관리
